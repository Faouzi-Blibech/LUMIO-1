What is LUMIO?

LUMIO is an AI-powered ADHD early detection and learning support platform designed for Tunisian school children. It connects students, teachers, and parents in a unified real-time ecosystem to identify attention patterns and deliver personalized learning interventions.

Core Value Proposition

  • Real-time focus monitoring via computer vision — no video transmitted (privacy-first)
  • Intelligent distraction analysis — understand why students lose focus, not just if
  • Role-based AI recommendations — tailored guidance for teachers, parents, and students
  • Professional referral detection — flags when specialist consultation may be needed
  • Homework struggle tracking — automatic detection of academic difficulties

Technical Foundation

Real-Time Data Pipeline

Browser-Side Computer Vision: MediaPipe FaceMesh runs locally. Computes focus_score from gaze, blink rate, head pose. Zero privacy risk — no video leaves the browser.
Real-Time Ingestion: Focus scores arrive via WebSocket every 1 second. Instant Redis cache + class-level broadcast for live teacher dashboards.
ML Classification: XGBoost categorizes why focus dropped — Fatigue, Difficulty, Environment, or Unknown.
Risk Profiling (Nightly): RandomForest scores student risk (0–1). Aggregates 30-day data. Outputs risk tier + flags persistent patterns.
AI Recommendations: Rule engine maps context to archetype. FAISS retrieves clinical docs. Groq LLM generates role-specific suggestions with safety rails.

Key Safety Features

🔒 Privacy & Security

  • ✓ No video/image transmission (browser-side CV only)
  • ✓ JWT tokens in httpOnly cookies (XSS-resistant)
  • ✓ Role-based data filtering (parents never see raw risk scores)

🛡️ AI Safety

  • ✓ Hardcoded blacklist: No diagnosis language allowed (ADHD, disorder, condition, autism)
  • ✓ Two-strike retry: LLM gets second chance with stricter prompt
  • ✓ Static fallback: Safe pre-written response if LLM fails twice
  • ✓ Rule engine supremacy: Deterministic logic always overrides AI on critical decisions

📋 Ethical Design

  • ✓ Support, not diagnosis — identifies patterns, never diagnoses
  • ✓ Professional referral only when persistent difficulties detected
  • ✓ Teachers make final intervention decisions (teacher-in-the-loop)
